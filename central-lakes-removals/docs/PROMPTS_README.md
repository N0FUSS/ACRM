## Purpose

These files define the strategic, visual, copy, SEO, CRO, trust, and development direction for the Central Lakes Removals website.

They should be used by coding agents, design agents, Figma agents, SEO agents, copy agents, and any person or system modifying the website.

## Core positioning correction

The commercial differentiator is not generic owner led moving.

The commercial differentiator is Russell Brown’s personal standard, judgement, care, lived expertise, accountability, and operating philosophy.

Central Lakes Removals is the delivery system built around those standards.

Owner involvement remains important, but it is supporting proof, not the main positioning idea.

## Homepage copy correction

The homepage must not repeat the same Russell Brown judgement message in every early section.

Each section has a different job:

1. Hero: create immediate trust through risk insight.
2. Proof bar: validate credibility quickly.
3. Russell difference: explain why expert judgement matters.
4. Services: route visitors to the right service.
5. Reviews: prove the experience through customer evidence.
6. Process: reduce uncertainty.
7. Premium value: justify paying properly.
8. Service areas: clarify geography truthfully.
9. About Russell: deepen the human trust anchor.
10. Quote section: convert.
