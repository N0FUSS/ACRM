# Central Lakes Removals — Project Memory

## Project Overview
Above-the-fold hero **prototype only** for **Central Lakes Removals**, a 30-year heritage furniture removals company based in Cromwell, Central Otago, NZ. Brand promise: *"Moving lives, not just furniture."* Brand pillars: Steady Hand (trust), Respect for the Journey (care), Eco-Conscious Protection, Good Humour / Great Service.

Source of truth for the brief: `/home/ubuntu/design_context.md` (synthesizes DESIGN.md + Brand Copy PDF + skill systems).

## Visual Style — "The Industrial Architect"
Editorial, mechanical-luxury, engineered-precision — never "friendly delivery."

- **Palette** (all dark, deliberately no pure white):
  - `--surface-container-lowest: #0a0e14`
  - `--surface: #10141a`
  - `--surface-container: #1c2026`
  - `--surface-container-highest: #31353c`
  - `--primary-gold: #f2ca50` (bright)
  - `--primary-container: #d4af37` (brushed deep gold)
  - `--on-surface: #f2ecdb` (warm off-white — NEVER #ffffff)
  - `--on-surface-variant: #d0c5af` (secondary text, uppercase labels)
- **Typography**: Montserrat ExtraBold (display, UPPERCASE, −2% tracking) + Open Sans 18px (body) + JetBrains Mono (micro-labels/serial numbers).
- **Corner radius**: ≤ 2px everywhere. No bubbly UI.
- **Shadows**: none by default. Depth comes from **tonal layering + glass + ghost borders**. Only critical modals get a 32–64px blur @ 8% opacity shadow tinted with `#0a0e14` — never pure black.

## CRITICAL Design Rules (zero-tolerance)
1. **NO solid line borders anywhere.** No `border-b`, no `<hr>`, no 1px full-width dividers. Section separation = surface-color shifts, whitespace rhythm (48 / 64 / 96 px), alternating surface stripes, or tonal gradients.
2. **Ghost borders only** — 1px gold @ 20% opacity, implemented as `inset box-shadow` (`.ghost-border` utility). Hover escalates to 50% (`.ghost-border-hover`).
3. **Glassmorphism only where semantically meaningful** — nav, modals, floating cards, hover menus. Recipe: `rgba(16,20,26,0.75)` + `backdrop-filter: blur(16px)`.
4. **No Inter, no purple gradients, no equal-3-column bento, no "Elevate your…" copy.**
5. Primary CTA: flat `#d4af37` with a 1px top bevel highlight (`.btn-bevel-top`), Montserrat ExtraBold UPPERCASE, 2px radius, no shadow.
6. Off-grid overlap must be **obviously intentional** (aligned to a 10% module).

## Tech Stack & Patterns
- **Next.js 14 App Router**, React 18, Tailwind 3, TypeScript strict.
- Theme is **forced dark** via `<html className="dark">` + `ThemeProvider forcedTheme="dark"` — do not re-enable light mode or a theme toggle.
- Fonts loaded via `next/font/google`: Montserrat → `--font-display`, Open Sans → `--font-sans`, JetBrains Mono → `--font-mono`.
- All brand tokens live in `app/globals.css` as CSS custom properties; Tailwind arbitrary values (`bg-[var(--surface)]`) consume them.
- Reusable glass / bevel / grain utilities defined in `globals.css @layer utilities`: `.glass-surface`, `.glass-surface-strong`, `.glass-surface-soft`, `.glass-reflection`, `.ghost-border`, `.bevel-top`, `.btn-bevel-top`, `.grain-overlay`, `.bg-video-vignette`, `.bg-focal-glow`, `.text-gold-gradient`, `.dot-separator`, `.scroll-cue-dot`, `.pulse-gold`, `.float-drift`, `.will-reveal`.
- **Tailwind arbitrary-value gotcha**: avoid `transition-[a,b]` with commas and `duration-[Nms]` — the JIT parser flagged them as ambiguous. Use `transition-all` + `duration-300` OR fall back to inline `style={{ transition: '...' }}`.

## Page / Component Structure
- `app/page.tsx` — server component, renders `<HeroSection />`.
- `app/layout.tsx` — server component, loads fonts, sets metadata (incl. `favicon.svg`, `og-image.png`), includes the `appllm-lib.js` script in `<head>`, forces dark theme.
- `components/hero/hero-section.tsx` — **client** orchestrator. 12-col grid: copy zone 7 cols / tracker zone 5 cols (with `lg:-ml-[10%]` overlap). Houses toasts for CTA interactions.
- `components/hero/video-background.tsx` — **client**. Renders `/videos/hero-truck.mp4` (the `Scenic Truck Drive Video.mp4` asset) with autoplay/muted/loop/playsInline, a parallax scroll transform, a video-vignette overlay, a focal-glow overlay, and a grain overlay. Video fade-in uses inline style (not Tailwind arbitrary duration).
- `components/hero/glass-nav.tsx` — **client**. Fixed top glass nav, CLR monogram + wordmark, smooth-scroll anchor links, phone tel: link, primary "Get a Quote" button, mobile drawer.
- `components/hero/tracker-card.tsx` — **client**. The off-grid floating Logistics Tracker glass card with animated gold-gradient progress spine, 4 steps (Pre-move Survey → Eco Packing → In Transit → Unload), 5-star footer.
- `components/hero/scroll-cue.tsx` — **client**. Animated scroll dot that smooth-scrolls to next section.
