# Central Lakes Removals Homepage Coding Agent Prompt

## 1. Purpose of this document

Use this document to design or build the Central Lakes Removals homepage.

It contains the homepage section order, copy, layout logic, component requirements, mobile priorities, visual rules, and coding agent instructions.

## 2. Homepage objective

The homepage must make one idea instantly clear:

> Your move, personally led by the owner.

The page must convert serious moving enquiries by building trust in Russell Brown, explaining the value of owner led moving, showing proof, and guiding visitors to request a quote.

## 3. Visual style

Use:

1. Dark midnight blue or near black background
2. Generous negative space
3. Warm off white typography
4. Muted grey supporting text
5. Brassy gold CTAs
6. Restrained brass accents
7. Real imagery of Russell
8. Strong editorial typography
9. Premium, calm layout rhythm

Do not create a generic moving company website.

## 4. Global layout system
